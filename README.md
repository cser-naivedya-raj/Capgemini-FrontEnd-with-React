## 📋 About This Repository

This repository contains all code, exercises, and mini-projects completed during the **Capgemini Front-End with React** training bootcamp. Each folder represents a single training day, organized chronologically with progressively advancing concepts — from raw HTML/CSS fundamentals to React component-based architecture.

**Tech Stack:** JavaScript · CSS · HTML · React

## 🛠️ Getting Started

To explore any day's work locally:

```bash
# Clone the repository
git clone https://github.com/cser-naivedya-raj/Capgemini-FrontEnd-with-React.git

# Navigate into any day's folder
cd Capgemini-FrontEnd-with-React/Day1-25-04-2026

# For React-based days, install dependencies and start
npm install
npm start
```

> **Note:** Plain HTML/CSS/JS files (Day 1 & 2) can be opened directly in a browser without any build step.

## 🎯 Learning Objectives

By the end of this training program, the goal is to be proficient in:

- Building structured, semantic HTML pages
- Styling responsive UIs with modern CSS (Flexbox, Grid)
- Writing clean, modern JavaScript (ES6+)
- Building React applications with functional components
- Managing component state with React Hooks (`useState`, `useEffect`)
- Understanding the React component lifecycle and data flow (props)
